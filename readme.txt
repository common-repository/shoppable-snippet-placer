== Description ==

A plugin to place the Shopsuite snippets correctly within Wordpress texts (This is the hexadecimal code found in
dashboard -> Widgtes.)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's all!

== Frequently Asked Questions ==

= Where can I find the hexadecimal code? =

Within your Shopsuite Publisher Dashboard Account, via snippets and code.
